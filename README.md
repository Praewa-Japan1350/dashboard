# Student Tracker Dashboard 🎓

ระบบ Dashboard อย่างง่ายสำหรับติดตามผลการเรียนและการเข้าเรียนของนักเรียน

### 🚀 วิธีการติดตั้งและรันโปรแกรม (How to run)
1.  **Clone Repository:**
    ```bash
    git clone <URL-ของ-Git-คุณ>
    ```
2.  **ติดตั้ง Library:**
    ```bash
    pip install -r requirements.txt
    ```
3.  **รัน Dashboard:**
    ```bash
    streamlit run app.py
    ```

### 📊 รายละเอียด Dashboard
- **Graph 1:** เปรียบเทียบคะแนนกลางภาคและปลายภาค (Bar Chart)
- **Graph 2:** แสดงอัตราการเข้าเรียนของนักเรียน (Line Chart)
- **Graph 3:** วิเคราะห์ความสัมพันธ์ระหว่างการเข้าเรียนและคะแนน (Scatter Plot)
- **Interactivity:** สามารถเลือกกรอง (Filter) ข้อมูลตามสาขาวิชา (Major) ผ่าน Sidebar ได้
